Objectness Threshold

Filters weak detections. A bounding box only appears if its confidence score meets this value.

Lower → more detections Higher → only confident ones

NMS Threshold

Non-Maximum Suppression merges duplicate boxes on the same object. Boxes with IoU above this value are merged.

Lower → aggressive merging Higher → overlapping boxes survive

Typical Settings

General road: obj=0.40, nms=0.45
High recall (safety): obj=0.25, nms=0.50
High precision: obj=0.60, nms=0.40
